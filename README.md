# Unilever Pricing Analytics Pipeline

## Project Overview
End-to-end pricing analytics project simulating a real Unilever commercial analytics workflow.The project integrates Excel, SQL, Python, statistics, and machine learning to analyze pricing strategy, margins, and regional performance.

## Key Objectives
- Assign price tiers based on brand strength
- Calculate recommended prices using margin logic
- Compare pricing strategy across regions
- Apply statistical testing and machine learning for prediction

## Tech Stack
- Excel (raw/ Unilever_products, finance targets)
- Python (Pandas, NumPy, Matplotlib, SciPy, Scikit-learn)
- Machine Learning (Regression & Classification)
